Sampling methods

Sources of sampling bias

Convenience sample individuals easily accessible are more likely to be included in sample
Non-response only a non-random fraction of sample respond survey
Volutanry response sample only on volunteer; because they have strong opinions on issue
Anecdotal based on anecdotal evidence; while maybe real, is based on a limited sample size that might not be representative of the population

Sampling methods

simple random sample (SRS)
  • randomly select cases from population
  • each case is equally likely to be selected
stratified sample
  • divide population into homogeneous groups: strata
  • randomly sample from within each stratum
cluster sample
  • divide population into clusters
  • randomly sample a few clusters
  • randomly sample within these clusters
  • each cluster similar to another; not necessarily homogeneous
  • usually did for economical reasons
