Overview

What is statistical inference?

  • the process of drawing conclusions from data
  • with data subject to random variation (observational errors or sampling)
  • need to quantify uncertainty associated with conclude from incomplete data that may change over time
  • drawing conclusions → inferring facts
  • create new knowledge from noisy uncertain data

in practice

  • used to test hypotheses and make estimations from sample data:
  • answers the questions: what should be done next?
  • descriptive statistics describe a sample; inferential statistics infer predictions about populations

Statistical propositions

  • statistical proposition: conclusion of a inference system
  • inference system or inductive system should produce reasonable and general answers
  • forms of statistical proposition:
estimate value that best approximates some parameter of interest
confidence interval or set estimate; set of values that would contain the true parameter value with some probability
credible interval or Bayesian confidence interval; set of values containing, for example, 95% of posterior belief
hypothesis test rejections of it
clustering or classification; arrange data points into groups

Styles of inference

Different thinking about probability leads to different styles of inference.

Frequency:
probability long run proportion of times an event occurs; independent, identically distributed repetitions
inference uses frequency probability interpretations to control error rates; "What should I decide given my data controlling the long run proportion of mistakes I make at a tolerable level."
Bayesian:
probability probability calculus of beliefs; given that beliefs follow certain rules
inference uses Bayesian probability representation of beliefs to perform inference; "Given my subjective beliefs and the objective information from the data, what should I believe now?"

Concerns to perform inference

  • Is the sample representative of the population that we'd like to draw inferences about?
  • Are there known and observed, known and unobserved or unknown and unobserved variables that contaminate our conclusions?
  • Is there systematic bias by missing data or the design or conduct of the study?
  • What randomness exists in the data and how do we use or adjust for it? —here randomness can be either explicit via randomization or random sampling, or implicit as the aggregation of many complex unknown processes.
  • Are we trying to estimate an underlying mechanistic model of phenomena under study?. If so we can build that model into our statistical inference probability model.
