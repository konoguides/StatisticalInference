Conditional probability

Conditional probability

Le \(B\) be an event so that \(P(B) > 0\). Then the conditional probability of an event \(A\) given that \(B\) has occurred is: $$P(A | B) = \frac{P(A \cap B)}{P(B)}$$ if \(A\) and \(B\) are not independent.

Event independence

Event \(A\) is independent of event \(B\) if: $$P(A|B) = P(A) \, \, \, \text{where} \, \, \, P(B)>0$$ equivalent to: $$P(A \cap B) = P(A)P(B)$$

IID random variables

  • said to be IID if they are independent and identically distributed
  • independent: statistically unrelated from one and another
  • identically distributed: all having been drawn from the same population distribution
  • are the default model for random samples

Rolling conditional dice

The probability of getting a 1 rolling a standar dice is assumed to be \(\frac{1}{6}\). Suppose you're given the extra information that the die roll was and odd number, hence 1, 3, or 5.

Conditional on this new information, what is the probability of getting a 1?

$$A=\{1\}; \, \, \, \, B=\{1,3,5\}$$ $$A \cap B = \{1\}$$

$$P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{P(\{1\})}{P(\{1,3,5\})} = \frac{1/6}{3/6} \,\,\, \therefore $$
$$P(\text{one given that roll is odd}) = P(A|B) = \frac{1}{3}$$

Baye's rule

  • allows to reverse the role of events involved in conditional probability
  • relates \(P(B|A)\) with \(P(A|B)\)

$$P(B|A) = \frac{P(A|B)P(B)}{P(A|B)P(B)+P(A|B^c)P(B^c)}$$ where:

  • \(P(A|B^c) = 1 - P(A^c|B^c)\)
  • \(P(B^c) = 1 - P(B)\)
