Conditional probability

Le \(B\) be an event so that \(P(B) > 0\). Then the conditional probability of an event \(A\) given that \(B\) has occurred, if \(A\) and \(B\) are not independent is: $$P(A | B) = \frac{P(A \cap B)}{P(B)}$$ but if \(A\) and \(B\) are independent, it becomes: $$P(A|B) = P(A) \, \, \, \text{where} \, \, \, P(B)>0$$

IID random variables

  • said to be IID if they are independent and identically distributed
  • independent: statistically unrelated from one and another
  • identically distributed: all having been drawn from the same population distribution
  • are the default model for random samples

Conditional probability rolling a dice

The probability of getting a 3 rolling a standar dice is assumed to be \(\frac{1}{6}\). Suppose you're given the extra information that the die roll was and odd number, hence 1, 3, or 5.

Conditional on this new information, what is the probability of getting a 3?

$$A=\{3\}; \, \, \, \, B=\{1,3,5\}$$ $$A \cap B = \{3\}$$

$$P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{P(\{3\})}{P(\{1,3,5\})} = \frac{1/6}{3/6} \,\,\, \therefore $$
$$P(\text{three given that roll is odd}) = P(A|B) = \frac{1}{3}$$

Baye's rule

  • allows to reverse the role of events involved in conditional probability
  • relates \(P(B|A)\) with \(P(A|B)\)
  • given that if \(A\) and \(B\) are not independent:

$$\begin{equation} P(A | B) = \frac{P(A \cap B)}{P(B)} \label{eq:one} \end{equation}$$ and, $$\begin{equation} P(B | A) = \frac{P(A \cap B)}{P(A)}, \label{eq:two} \end{equation}$$ from \eqref{eq:one} we an obtain: $$\begin{equation} P(A \cap B) = P(A | B) * P(B) \label{eq:three} \end{equation}$$ and substitude it on \eqref{eq:two}, we get: $$\begin{equation} P(B | A) = \frac{P(B)}{P(A)} P(A | B) \label{eq:four} \end{equation}$$ Please not that \(P(A)\) can be rewritted as: $$\begin{equation} P(A) = P(A \cap B) + P(A \cap B^c) \label{eq:five} \end{equation}$$ using \eqref{eq:three} on \eqref{eq:five} we have: $$\begin{equation} P(A) = P(A | B) * P(B) + P(A | B^c) * P(B^c) \label{eq:six} \end{equation}$$ finally changing \(P(A)\) from \eqref{eq:four} by \eqref{eq:six}, we end up with the Baye's rule

$$P(B|A) = \frac{P(A|B) * P(B)}{P(A|B) * P(B)+P(A|B^c) * P(B^c)}$$

where:

  • \(P(A|B^c) = 1 - P(A^c|B^c)\)
  • \(P(B^c) = 1 - P(B)\)
