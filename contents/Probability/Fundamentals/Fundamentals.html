Fundamentals

Probability, what is it?

  • is a measure of the likeliness that an event will occur
  • link the sample with the population
  • quantify attitude towards some proposition, whose truth we are not certain
  • numerical measure (between 0 and 1): 0 - impossibility, 1 - certainty
  • given a random experiment, probability is a population quantity summarizes the randomness

Random variable

  • numerical outcome of an experiment
  • possible values represent the possible outcomes of a experiment
  • or the "subjective" randomness that results from incomplete knowledge of a quantity
  • also called aleatory variable or stochastic variable
  • two principal varieties:
discrete random variable we talk about the probability that they take specific values
continuous random variable we talk about the probability that they line within some range

Sample space

  • set of all possible outcomes or results of a experiment or random trial
  • usually denoted using set notation
  • common to refer to a sample space by the labels \(S\) or \(\Omega\)
  • \(U\) for universal set

Event

  • set of outcomes of an experiment
  • subset of the sample space
  • has a probability assigned
  • complementary event (set): the event not occuring, defined for each event
  • Bernoulli trial: event + complementary event

Sum of probabilities

For any two events \(A\) and \(B\), the probability that at least one occurs: $$P(A \cup B) = P(A) + P(B)$$ if \(A\) and \(B\) are mutually exclusive, or: $$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$ if \(A\) and \(B\) are NOT mutually exclusive.

Probability model

  • also known as probabilistic space
  • construct that models a real-world process (or experiment)
  • link the sample with the population using assumptions
  • consist of three parts:
1. sample space set of all possible outcomes
2. set of events sigma algebra; each containing zero or more outcomes
3. probability measure function a probability assigned to each event; a function \(P\) from events to probabilities
