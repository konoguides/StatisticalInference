Hypothesis testing

What is hypothesis testing?

  • formal procedures used to reject or not statistical hypotheses
  • formally decide if something is likely or unlikely
  • allow to determine if something is statistically significant
  • evaluates two mutually exclusive statements about a population
  • determines which is best supported by random sample from population
  • if sample not consistent with the statistical hypothesis, the hypothesis is rejected
  • helps to determine if an intervention significantly changes a population prameter

Levels of likelihood \(\left( \alpha \right)\)

  • commonly known as alpha or significance levels
  • acts as a threshold or criteria to decide if something is likely or unlikely
  • address the problem it's difficult to decide upon a threshold what constitutes likely or unlikely
  • statisticians have determined three conventional levels of likelihood (or unlikelihood): $$\begin{array}{lrl} P(x) = 0.05 & (\alpha = 5\%) & \rightarrow & \text{unlikely} \\ P(x) = 0.01 & (\alpha = 1\%) & \rightarrow & \text{unlikely} \\ P(x) = 0.001 & (\alpha = 0.1\%) & \rightarrow & \text{very unlikely} \end{array}$$
  • this way, if probability of getting an observation is less than \(\alpha\), it is "unlikely" to occur

Significant results

Suppose we have a \(\text{z-score}\) for a sample mean \(\left( \bar{x} \right)\) of: $$\text{z-score} = \frac{\bar{x} - \mu}{SE} = 1.82,$$

We can only say \(\bar{x}\) is significant at \(p \lt 0.05\), because \(1.82 > 1.65\), the z-critical value for the \(\alpha = 0.05\) critical region, but \(1.82 \ngtr 2.32\) the z-critical value for the \(\alpha = 0.01\) critical region.

Constructing a hypothesis test

1. State the hypothesis
  • state null and alternative hypotheses
  • in mutually exclusive way
2. Formulate an analysis plan
  • describes how to use sample to evaluate the null hypothesis
3. Analyze sample data
  • find the value of the test statistic
  • describen in analysis plan
4. Interpret results
  • apply the decision rule described in the analysis plan
  • if test statistic's value is unlikely → reject the null hypothesis
Decision rules
  • rules for rejecting the null hypothesis
  • approaches are equivalent
P-value
  • strength of evidence in support of a null hypothesis
  • if P-value \(< \alpha\) → reject null hypothesis
  • probability of observing a test statistic as extreme as calculated from sample, assuming the null hypotheis is true
Region of acceptance
  • range of values
  • if test statistic falls in → null hypothesis is not rejected
  • region of rejection: range of values outside the region of acceptance
Decision errors
Type I error
  • occurs when rejects a null hypothesis that is true
  • significance level \(\left( \alpha \right)\): probability of commiting it
Type II error
  • occurs when fails to reject a null hypothesis that is false
  • Power: probability of not commiting it
  • Beta \(\left( \beta \right)\): probability of commiting it

One-tailed & two-tailed tests

One-tailed test
  • used when predicting the direction of the treatment effect
  • region of rejection on only one side of sampling distribution, e.g.
$$\begin{array}{rl} H_0 : & \mu \geq \mu_I \\ H_1 : & \mu \lt \mu_I \end{array}$$
$$\begin{array}{rl} H_0 : & \mu \leq \mu_I \\ H_1 : & \mu > \mu_I \end{array}$$
Two-tailed test
  • non-directional hypothesis test
  • more conservative: less likely to reject the null when it's true
  • region of rejection on both sides of sampling distribution
  • \(\frac{\alpha}{2}\) on each side, e.g.
$$\begin{array}{rl} H_0 : & \mu = \mu_I \\ H_1 : & \mu \neq \mu_I \end{array}$$
