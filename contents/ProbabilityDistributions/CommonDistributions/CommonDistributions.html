Normal distribution (norm)
Notation: \(\mathscr{N}\left(\mu, \sigma^2\right)\)
Parameters \(\mu \in \mathscr{R}\) — mean
\(\sigma^2 > 0\) — variance
PDF: $$f \left( x | \mu, \sigma^2\right) = \frac{1}{\sigma \sqrt{2 \pi}} e^{-\frac{\left( x-\mu \right)^2}{2 \sigma^2}}$$ CDF: $$F_X \left( x | \mu, \sigma^2\right) = \frac{1}{2} \left(1 + erf\left(\frac{x-\mu}{\sigma \sqrt{2}}\right)\right)$$ 
from scipy.stats import norm
# KEYWORDS: loc = mean, scale = standard deviation
ndst = norm(loc=0, scale=1)         # create a frozen distribution
mean, var, skew, kurt = norm.stats(loc=0, scale=1, moments='mvsk')
mean, median, var, std = ndst.mean(), ndst.median(), ndst.var(), ndst.std()
# PDF
norm.pdf(x, loc=0, scale=1)
# CMF
norm.cdf(x, loc=0, scale=1)
# Survival function
norm.sf(x, loc=0, scale=1)
# Percent point function (inverse of cdf — percentiles)
norm.ppf(q, loc=0, scale=1)
# Inverse survival function
norm.isf(q, loc=0, scale=1)
# Endpoints of the range that contains alpha
# percent of the distribution, simetricaly from the mean
norm.interval(alpha, loc=0, scale=1)
# Random variates
norm.rvs(loc=0, scale=1, size=1, random_state=None)
# Parameter estimates (mean, std) for generic data
norm.fit(data)
# Expected value of a function (of one argument)
# with respect to the distribution
norm.expect(func, loc=0, scale=1, lb=None, ub=None,
            conditional=False, **kwds)
Student's T distribution (t)
Parameters \(\nu > 0\) — dof
Mean:
Median: 0
Mode: 0
Variance:
PDF: $$f \left( x | \nu\right) = \frac{\Gamma \left(\frac{\nu + 1}{2}\right)}{\sqrt{\nu \pi}\Gamma \left(\frac{\nu}{2}\right)} \left( 1 + \frac{x^2}{\nu} \right)^{-\frac{\nu + 1}{2}}$$
CDF: $$F_X \left( x | \nu\right) = \frac{1}{2}+x\Gamma\left(\frac{\nu+1}{2}\right)\times\frac{_{2}F_{1}\left(\frac{1}{2},\frac{\nu+1}{2};\frac{3}{2};\frac{x^{2}}{\nu}\right)}{\sqrt{\nu\pi}\Gamma\left(\frac{\nu}{2}\right)}$$ where \(_{2}F_{1}\) is the hypergeometric function
