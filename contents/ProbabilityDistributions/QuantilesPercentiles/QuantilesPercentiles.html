Quantiles & percentiles

Quantiles

  • values taken at regular intervals from the CDF-inverse
  • motivation: dividing ordered data into essentially equal-sized subsets
  • values marking the boundaries between consecutive subsets

\(\alpha^{th}\) quantile

The \(\alpha^{th}\) quantile fo a distribution with cumulative distribution function \(F_X\) is the point \(x_{\alpha}\) so that $$F_X(x_{\alpha}) = \alpha$$

When a quantile come from a distribution it is refering to the population. When comes from sample it is the estimator of the population estimand.

Percentile

  • simply a quantile with \(\alpha\) expressed as percent

percentiles with Python

>>> import numpy as np
>>>
>>> data = [1, 3, 4, 2, 5, 6]
>>> np.percentile(data, 25)
2.25
>>> np.percentile(data, 50)
3.5
>>> np.percentile(data, 75)
4.75

Specialiced quantiles

Some q-quantiles have special names:

median the only 2-quantile sextiles → S the 6-quantiles
terciles → T the 3-quantiles deciles → D the 10-quantiles
quartiles → Q the 4-quantiles vigintiles → V the 20-quantiles
quintiles → QU the 5-quantiles percentiles → P the 100-quantiles
