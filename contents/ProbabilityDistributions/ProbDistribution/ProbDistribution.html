Probability distribution

Probability distribution

  • function \(P\) from random variable value to its associated probability
  • propose statements about the population, not of the data itself
  • random variates: realizations of random variable according to its probability distribution
  • can be univariate or multivariate
  • the sum of possible values (or area) has to add up to one
  • must be \(\geq 0\) everywhere
  • two principal varieties:
discrete for discrete random variables; known as probability mass function (PMF); probability that the variable takes a specific value
continuous for continuous random variables; known as probability density function (PDF); probability that variable line within some range; area under PDF correspond to probabilities for that random variable; derivate of CDF

Cumulative distribution function (CDF)

  • returns probability that random variable \(X\leq x\) some value $$F_X(x) = P(X \leq x)$$
  • it gives the area under the PDF associated with \(X\), from \(-\infty\) to \(x\)
  • also used to specify distribution of multivariate random variables
  • known simply as distribution function

Survival functions

  • returns the probability that random variable \(X \gt x\) value $$S_X(x) = P(X \gt x)$$
  • defined as the complement to one of CDF: $$S_X(x) = 1 - F_X(x)$$

How to specify a probability distribution

A probability distribution can be specified by suppluying:

  • a probability mass function (pmf)
  • a probability density function (pdf)
  • a cumulative distribution function
  • a survival function
  • a hazard function
  • a characteristic function
  • a rule for constructing a new random variable from other random variables, whose joint probability distribution is known
