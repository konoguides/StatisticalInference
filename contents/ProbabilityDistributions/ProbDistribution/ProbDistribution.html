Probability distribution

A function \(F\) that maps the value \(x\) of a random variable to its probability of occurrence. The area beneath the curve described by \(F\) must be 1, and \(P(x) \geq 0 \).

The proposed statements are about the population, not of the data itself, and can be univariate or multivariate.

Two principal varieties
PDF: probability density function continuous
  • probability that variable lie within some range
  • derivate of CDF
  • area under PDF → probabilities for that r.var
PMF: probability mass function discrete
  • probability that the variable takes a specific value

Cumulative distribution function (CDF)

  • known simply as distribution function \(\left(F_X\right)\): returns probability that random variable \(X\leq x\) $$F_X(x) = P(X \leq x)$$
  • gives area under PDF associated with \(X\), from \(-\infty\) to \(x\) $$F_X(x) = \int_{-\infty}^x PDF(x)\cdot x \cdot dx$$
  • also used to specify distribution of multivariate random variables

Survival functions (SDF)

  • returns the probability that random variable \(X \gt x\) $$S_X(x) = P(X \gt x)$$
  • defined as the complement to one of CDF: $$S_X(x) = 1 - F_X(x)$$

How to specify a probability distribution

A probability distribution can be specified by suppluying:

  • a probability mass function (PMF)
  • a probability density function (PDF)
  • a cumulative distribution function (CDF)
  • a survival function (SDF)
  • a hazard function
  • a characteristic function
  • a rule for constructing a new random variable from other random variables, whose joint probability distribution is known
